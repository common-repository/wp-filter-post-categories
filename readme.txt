Filters post categories on your pages

== Description ==
This plugin allows you to choose which post categories youe site will show on your pages. Just go to settings and deselect the categories that you want to hide.
For the homepage it will work automatically, for the other pages you have to add a shortcode into the page.

== Installation ==
1. Upload `merlic-wp-filter-post-category.zip` from the plugin panel or unzip the file and upload the folder `merlic-wp-filter-posts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` panel in WordPress
3. Go to `Settings->Filter Posts in Pages` to choose the categories to show/hide from the frontpage

If you get errors or it does not work, check the `merlic-wp-filter-posts` plugin folder permissions. They should be 755.
Plugin requires php 5.2


== Frequently Asked Questions ==

= Is it possible to assign categories to other pages? =

Yes. Go the the settings page, select the categories that you want to show and copy the generated shortcode into your page in the spot where you want the posts to show up
